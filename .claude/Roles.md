# Werewolf Game Roles

This document catalogs all available roles in the Werewolf (狼人杀) game, organized by alignment. Roles marked with ✓ are implemented; unmarked roles are available for future implementation.

---

## Good Roles (村民阵营)

| Role Name | Chinese | Implemented | Description | Night Action | Once Per Game | Special Notes |
|-----------|---------|--------|-------------|--------------|----------------|----------------|
| Villager | 平民 | ✓ | No special abilities. Must deduce the truth from mixed information. | No | | |
| Seer | 预言家 | ✓ | Can see a player's alignment (werewolf or good) at night. | Yes | | |
| Witch | 女巫 | ✓ | Has two potions: one to save a player killed by werewolves, one to poison any player. Can use only one potion per night. | Yes | | Witch can save herself depending on game host setting |
| Dreamcatcher | 摄梦人 | ✓ | Each night, makes a player a "sleepwalker," rendering them immune to all night kills. If the dreamcatcher dies, the sleepwalker dies too. | Yes | | |
| Hunter | 猎人 | ✓ | If killed by werewolves or executed, can shoot and kill any player before dying. | No | | Cannot shoot if killed by poison or other special means |
| Guard | 守卫 | ✓ | Each night, protects one player from being killed by werewolves. Cannot guard the same person two nights in a row. Can guard themselves. | Yes | | |
| Magician | 魔术师 | ✓ | Switch two players' role cards. | Yes | | |
| Demon Hunter | 猎魔人 | ✓ | Picks a player to eliminate. If the player is a werewolf, the werewolf is eliminated. If the player is good-aligned, the demon hunter is eliminated. | Yes | | Starting from second night can activate ability. Immune to witch poison |
| Fool | 白痴 | ✓ | If voted out, reveals their identity and survives, but loses voting rights. Still can be killed by werewolves. | No | | |
| Knight | 骑士 | ✓ | Before voting, can reveal their identity and accuse a player. If correct (the player is a werewolf), that player dies. If wrong, the knight dies. | No | | |
| Gravedigger | 守墓人 | ✓ | Knows the alignment of the person who was voted out during the day phase. | No | | |
| Crow | 乌鸦 | ✓ | Each night, curse a player to receive 1 extra vote during exile voting tomorrow. | Yes | | Cannot curse the same player two nights in a row. Can skip. Can target self. |
| Miracle Merchant | 奇迹商人 | ✓ | Each night, give witch poison, seer ability, or guard ability to another player. If the target is a werewolf, the Merchant is eliminated the next day instead. | Yes | | Recipient can use their granted ability immediately that same night. |
| Sequence Prince | 定序王子 |  | Can reveal themselves and activate their skill to rewind time to just before the previous vote, allowing the village to vote again. | No | Yes | |
| White Maiden | 纯白之女 | ✓ | Checks a person's alignment. If it is a werewolf, that werewolf is immediately eliminated. | Second Night | | Witch cannot save a player eliminated by her check. Guard shield cannot protect against her check |
| Alchemy Witch | 炼金魔女 |  | At night, can use "Unidentified Mist" to select three different players. The werewolf team must choose their attack target from among these three players. Can also use "Pharaoh's Serpent" to save a player from being eliminated before the next day's elimination is announced. | Second Night | Yes | Each ability can only be used once per game. Can only save players attacked by werewolves, not those eliminated by other special roles |
| Child Fox | 子狐 |  | From the second night onward, can choose to "confuse" one player during the night phase. If the chosen player is a werewolf, the entire werewolf team cannot use their "kill" ability that night. If a "Wolf Beauty" is confused, they cannot use their charm ability that night. Confusing a good person has no effect. | Yes | Yes | Cannot confuse the same player two nights in a row |
| White Cat | 白猫 |  | When eliminated, the White Cat flips their card but does not leave the game immediately. Remains in play and only leaves after the next voting phase is completed. | No | | |
| Pufferfish | 河豚 |  | After a voting round, can reveal their identity and immediately eliminate all players who voted for them in that round. | No | Yes | If killed by werewolves at night, they reveal themselves the next day and disable "Wolf Beauty" ability for that day |
| Bear | 熊 |  | At dawn, if alive and at least one of the two players sitting directly next to the Bear is a werewolf, the judge announces: "Bear roared." If dead or neither neighbor is a werewolf, announces: "Bear did not roar." | No | | |
| Alpaca | 羊驼 |  | No special skills. | No | | Essentially a Villager |
| Day Scholar | 白昼学者 |  | Can choose a player to "amplify" (use special ability one extra time that night) or "weaken" (cannot use special ability that night). If weakening a werewolf, the entire werewolf team cannot attack that night. | Second Night | | Can only use either amplify or weaken each night, not both. Guard's extra protection from amplification does not need to follow consecutive protection restriction |
| Count of Light | 流光伯爵 |  | The protected player is immune to all night-time damage, including attacks from werewolves, poison from the Witch, and harm from the Dreamcatcher. | Second Night | | Cannot protect the same player two nights in a row |
| Silence Elder | 禁言长老 |  | | | | |
| Mirror Girl | 魔镜少女 |  | Sees a person's specific role (exact identity, not just alignment). | Yes | | |
| Awakened Hunter | 觉醒猎人 |  | When eliminated, can immediately choose to use their special skill "Hunt" to select the next player in the seating order (clockwise or counterclockwise). If that player is a werewolf, eliminate them from the game. | No | | |
| Awakened Seer | 觉醒预言家 |  | Each night, can choose any two players to inspect their exact identities and determine player alignment. | Yes | | |
| Awakened Lonely Girl | 觉醒孤独少女 |  | At the start of the first night, chooses one player to become her idol. If the idol is eliminated by voting, the Lonely Girl transforms into a werewolf and joins the wolf team. If the idol dies by any other means, the Lonely Girl inherits the idol's role and abilities. | Yes | | |

## Bad Roles (狼人阵营)

| Role Name | Chinese | Implemented | Description | Night Action | Once Per Game | Special Notes |
|-----------|---------|--------|-------------|--------------|----------------|----------------|
| Werewolf | 狼人 | ✓ | Chooses a player to kill each night with other werewolves. Can self-destruct during the day to instantly turn into night phase. | Yes | | Werewolves can see each other and discuss during night phase |
| Wolf King | 狼王 | ✓ | When eliminated (either at night or by voting), can immediately choose another player to be eliminated along with them. | Yes | | If Wolf King reveals identity on their own (self-exposes), cannot use this ability. If eliminated by Witch's poison, cannot use ability either |
| White Wolf King | 白狼王 |  | A werewolf who, when self-destructing during the day, can take another player with them. | Yes | | If eliminated in any other way, cannot use this skill. Neither they nor the taken player gets to make final statements when this ability is used |
| Ghost Rider | 恶灵骑士 |  | Immunity to night kills: cannot be killed at night by werewolves or witch poison. If the Seer checks the Ghost Rider, the Seer is eliminated the next day. If the Witch poisons them, the Witch is eliminated the next day. | Yes | Yes | Cannot self-destruct or attack themselves. Counterattack ability can only be triggered once per game. Immunity to night kills is permanent, but counterattack is one-time only |
| Wolf Beauty | 狼美人 | ✓ | Each night, after participating in the werewolf attack, can secretly "charm" one player. If Wolf Beauty is eliminated (by voting or night kill), the charmed player from the previous night is also eliminated. The charmed player does not know they have been charmed. | Yes | | If eliminated by a knight's duel, the ability does not activate. Cannot self-destruct or self-kill |
| Gargoyle | 石像鬼 |  | Each night, can check the exact identity of one player (more powerful than the Seer). If all other werewolves are eliminated, the Gargoyle can attack at night. | Yes | | Cannot self-destruct, cannot discuss with other werewolves at night, cannot participate in attacks with them, and does not know the identities of other werewolves |
| Hidden Wolf | 隐狼 |  | Always appears as a "good person" (villager) when checked by the Seer. Cannot attack at night while other werewolves are still alive. Only after all other werewolves are eliminated can the Hidden Wolf perform a night attack. | Yes | | Knows the identities of werewolf teammates but does not know their specific roles. Other werewolves do not know who the Hidden Wolf is. Cannot self-destruct |
| Blood Moon Apostle | 血月使徒 |  | On the night following their self-destruction, all abilities of the "good" roles (such as Seer, Witch, Hunter, etc.) are sealed and cannot be used. If the Blood Moon Apostle is the last werewolf to be exiled, they survive until the next daybreak before being eliminated. | No | | |
| Nightmare Shadow | 噩梦之影 |  | Each night, before anyone else acts, can choose one player to "fear," making them unable to act that night. If chosen player is a werewolf, the entire werewolf team cannot attack that night. Same player cannot be targeted two nights in a row. On first night, does not know other werewolves' identities and may accidentally target a teammate. | Yes | | Can choose to skip using ability. If used on Hunter and Hunter is voted out, Hunter can still shoot. If used on Hunter attacked by werewolves, Hunter cannot shoot next day |
| Time Wolf Queen | 蚀时狼妃 |  | Each night, can wake up first and choose any player to be locked in a time rift. If good-aligned players use investigation, poison, or protection abilities on the locked player that night, those abilities are reflected back onto themselves. After the ability is triggered, the Time Wolf Queen permanently loses this skill. | Yes | Yes | Witch's antidote cannot be reflected. Multiple complex interactions with Guard and Witch protection when both target the locked player |
| Wolf Witch | 狼巫 | ✓ | Each night, can check the exact role of one player who is not on the werewolf team. | Yes | | Check happens after the werewolves attack. Special interaction: if Pure White Maiden eliminates the last werewolf (Wolf Witch) and Wolf Witch eliminates the Pure White Maiden on same night, werewolf team wins |
| Wolf Raven Claw | 狼鸦之爪 |  | In a sealed state at game start. When living werewolf team members drop below 3, the seal breaks: learns other werewolves' identities, joins discussions and attacks. After other werewolves finish night actions, can make a solo attack that ignores all protection effects. | Yes | | At start: cannot participate in discussions, cannot attack, cannot know other werewolves. Other werewolves know who this role is. After unsealing, can make one extra attack that bypasses protection |
| Silent Night Mentor | 寂夜导师 |  | Can amplify (use ability one extra time) and weaken (cannot use ability) other players at night, similar to Day Scholar. Cannot participate in werewolf discussions or attacks at night, nor know other werewolves' identities at first, but other werewolves know who they are. | Second Night | | If both Silent Night Mentor and Day Scholar target same player, both abilities cancel and the target is eliminated. If both target a werewolf, both abilities cancel and the Mentor is eliminated. Weakening Witch allows them to still use antidote |
| Sun Eclipse Maid | 蚀日侍女 |  | Acts first at night and chooses one non-werewolf player to "devour." Gains that player's skill and inherits its current state, allowing her to use it that same night. The devoured player loses their skill for that night and cannot use it until the next day. | Second Night | | Devouring certain special roles results in backlash and elimination next day. Knows other werewolves' identities, but they do not know hers. Cannot participate in discussions or attacks, and cannot self-destruct. If all other werewolves are eliminated, can attack at night |
| Night Noble | 夜之贵族 |  | Can choose one other player to become their "Night Servant." The Night Servant will be eliminated at the next dawn. Only Night Noble knows who the Night Servant is. Original abilities, status, and team of the chosen player are not affected until elimination. | Second Night | | Elimination can be blocked by "Secret Identity" ability. Can use skill on a werewolf to protect them. If Witch Hunter hunts a Night Servant, neither is eliminated next day and Night Servant status is removed |
| Awakened Wolf Beauty | 觉醒狼美人 |  | Can use special skill "Elegy Illusion" to charm one player while werewolves open their eyes at night. If the Awakened Wolf Beauty is about to be eliminated for the first time, the charmed player is eliminated instead. | Yes | Yes | Can use skill once every other night, but effect only works once per game. Part of wolf team and cannot reveal identity or attack themselves |
| Awakened Gargoyle | 觉醒石像鬼 |  | Wakes up alone and selects one player adjacent to the werewolf faction to become a Converter. The Converter's faction changes to werewolf but does not meet with the wolf team and cannot publicly reveal identity. Before other werewolves are eliminated, Converter can use original abilities. After, loses original abilities and uses claw attack skill at night. | Yes | | On first night, Seer sees Converter as good team. On second night, Seer sees them as bad team. If no one is selected, random adjacent player is selected |
| Awakened Hidden Wolf | 觉醒隐狼 |  | May choose one player to imitate, gaining that player's identity and their abilities after that night ends. Different imitations grant different bonuses: werewolf gains extra one-time kill, Magic Mirror Girl gains nightly identity check, Witch gains poison, Guard gains protection, Hunter gains post-death elimination ability. | Yes | | Cannot publicly reveal identity. Cannot participate in wolf team discussions or attacks at night or know other wolves' identities. Other wolves don't know them. After other wolves eliminated, can participate in night kill |
| Awakened Wolf King | 觉醒狼王 |  | Has two Wolf King Claw abilities. At night, can pass one Wolf King Claw to any living wolf teammate. A player with a Wolf King Claw can choose one player to eliminate with them after being eliminated. Knows all teammate identities at game start and participates in night attacks. Only one Claw can be passed each night. | Yes | | Cannot pass to eliminated teammates, themselves, or good faction players. If self-stabs at night and dies, can still use claws. If dies from poison or dream-related death, cannot use claws. If passes Claw to teammate, teammate can still use it even if Wolf King dies from poison |
| Scent Phantom | 寻香魅影 |  | Bind two players together. If one person dies, the other dies too. Can use ability every night and can even include themselves in the bind. If scent phantom dies, the bind stays for one more night. | Yes | | Cannot reveal themselves. Binding happens after werewolves choose someone. If bound targets include Hunter or Awakened Wolf King, they cannot use their death abilities to take someone with them |

## Neutral Roles (中立阵营)

| Role Name | Chinese | Implemented | Description | Night Action | Once Per Game | Special Notes |
|-----------|---------|--------|-------------|--------------|----------------|----------------|
| Cupid | 丘比特 |  | On the first night, chooses two players to become lovers. If one dies, the other dies as well. If lovers are from different factions, their win condition changes to surviving together. | Yes | Yes | |
| Cursed Fox | 咒狐 |  | Cannot be eliminated by werewolf attack; survives if attacked. However, if the Seer investigates the Cursed Fox, they are immediately eliminated. If either werewolves or villagers achieve their victory condition while the Cursed Fox is alive, the Cursed Fox overrides the win and becomes the sole winner instead. | No | | Seer sees Cursed Fox as "good person" but they are still eliminated if checked |
| Secret Admirer | 暗恋者 |  | At the start of the game (first night), selects one player as their secret crush. Win condition is tied to the fate of the chosen player: if the admired player wins, the Secret Admirer also wins, regardless of which side that player is on. | Yes | | For all investigative purposes (like the Seer), always appears as "good guy" (villager). Cannot choose themselves as target. If no choice in time, system randomly assigns a target |
| Chameleon | 千面人 |  | | | | |
| Wild Child | 野孩子 |  | Chooses a role model on the first night. If the role model dies, becomes a werewolf and joins the werewolf team. | Yes | Yes | |
| Bomber | 炸弹人 |  | If voted out (banished) during the day, every player who voted for the Bomber will immediately die. | No | Yes | If Bomber dies by any other means (night kill, etc.), ability does not activate. If all other players are killed by this effect when Bomber is banished, the Bomber wins the game alone |

## Notes

- **Night Actions**: Some roles have abilities that activate during specific phases (e.g., "Second Night" means the ability is available starting from the second night)
- **Once Per Game**: Marked with "Yes" if the role has an ability that can only be used once throughout the entire game
- **Role Interactions**: Many roles have complex interactions with other roles—refer to the "Special Notes" column for important interaction details
- **Alignment**: Good roles win with the village, Bad roles win with the werewolves, Neutral roles have independent win conditions
